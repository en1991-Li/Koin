Koin記帳
Koin 是一款專為追求效率與視覺美感的用戶設計的記帳與財務管理工具。透過直觀的圖表與帳戶分組功能，幫助你一目了然地掌握資產、負債與每日收支細節。

🚀 核心功能
帳戶總覽 (Accounts Overview)：整合資產與負債數據，提供總額、資產與負債的即時數據面板。

動態趨勢圖表：內建資產趨勢與分組佔比圓餅圖，視覺化呈現財務健康狀況。

彈性的帳戶管理：

支援多種帳戶類型（現金、銀行、數位銀行、信用卡等）。

信用帳戶專屬功能：可設定信用額度、繳款期限及帳單週期。

詳細記帳系統：

支援支出、收入、轉帳、應收/應付等多種記帳類型。

內建常用商家快捷圖示（如：7-11、麥當勞、全家、Gogoro 等）。

日曆視圖：以日曆形式檢視每日收支，快速回顧財務軌跡。

專案統計：針對特定生活開銷、投資理財、旅遊等專案進行獨立預算與支出統計。

📝 邏輯說明
信用帳戶聯動：當開啟「信用帳戶」開關時，介面會動態顯示繳款期限與信用額度欄位，並將金額視覺效果轉為代表負債的紅色。

頁面導覽：採用單頁應用 (SPA) 邏輯，透過 showPage() 函數平滑切換各功能模組。
