# Koin記帳

**Koin記帳**是一款專為追求效率與視覺美感的用戶設計的記帳與財務管理工具。
透過直觀的圖表與帳戶分組功能，幫助你一目了然地掌握資產、負債與每日收支細節。

🚀 **核心功能**
- **帳戶總覽**：整合資產與負債數據，提供總額、資產與負債的即時數據面板。
- **動態趨勢圖表**：內建資產趨勢與分組佔比圓餅圖，視覺化呈現財務健康狀況。
- **彈性的帳戶管理**：
  - 支援多種帳戶類型（現金、銀行、數位銀行、信用卡等）。
  - 信用帳戶專屬功能：可設定信用額度、繳款期限及帳單週期。
- **詳細記帳系統**：
  - 支援支出、收入、轉帳、應收/應付等多種記帳類型。
  - 內建常用商家快捷圖示（如：7-11、麥當勞、全家、Gogoro 等）。
- **日曆視圖**：以日曆形式檢視每日收支，快速回顧財務軌跡。
- **專案統計**：針對特定生活開銷、投資理財、旅遊等專案進行獨立預算與支出統計。
  
🛠 **技術棧**
- **前端技術**：HTML5, CSS3 (包含深色模式優化)
- **交互邏輯**：JavaScript / ES6+
- **圖示庫**：Lucide Icons
- **版面佈局**：響應式設計 (Mobile-First)

📂 **專案結構** 

**index.html**：核心骨架與容器

定義 SPA（單頁應用）的所有 section 頁面結構。

配置全域導覽列（Tab Bar）與 FAB（浮動操作按鈕）。

收納所有互動彈窗（Modals），包含帳戶分組、週期選擇與繳款期限。

**style.css**：視覺規範與佈局

手機模擬器：設定 .phone-container 的寬高比與陰影，營造實機感。

深色模式：定義全域深色背景（#1c1c2e）與文字階層色彩。

自定義組件：包含 iOS 風格開關（Switch）、自定義滑桿（Slider）與滾輪選擇器（Picker）樣式。

**script.js**：核心調度與邏輯引擎

導覽控制：管理 showPage() 邏輯，處理頁面切換與 Lucide 圖示重新渲染。

帳戶管理：處理「新增帳戶」的資料驗證、信用帳戶欄位連動與數據儲存。

持久化層：封裝 localStorage 操作，確保使用者資料在重新整理後不消失。

**projects.js**：專案模組與動態渲染

資料模型：定義專案的預設數據與使用者新增數據的合併邏輯。

列表渲染：動態生成「專案總覽」列表，並根據類型（如統計專案）呈現不同樣式。

業務邏輯：處理「新增專案」頁面的數據寫入與即時同步。

**calendar.js**：日曆視圖與日期處理

日曆生成：動態計算並渲染 2026/04 月份的日期網格。

明細關聯：處理點擊日期後，下方收支明細列表的過濾與顯示邏輯。
